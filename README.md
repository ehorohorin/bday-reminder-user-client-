# Инструкция по запуску

## Бот для напоминаний о днях рождения в Telegram

## 1. Установка

Откройте терминал в папке с проектом и выполните:

```bash
pip install -r requirements.txt 
```

---

## 2. Создание проекта в Google Cloud Console

> Выполняется один раз. В результате вы получите файл `credentials.json` для доступа к контактам Google.

1. Перейдите на https://console.cloud.google.com
2. Вверху нажмите на выпадающий список проектов → **Создать проект** → введите любое название → **Создать**
3. В левом меню выберите:
   - **API и сервисы** → **Библиотека**
   - Найдите **People API**
   - Нажмите **Включить**
4. Перейдите в:
   - **API и сервисы** → **Учётные данные**
   - **Создать учётные данные** → **Идентификатор клиента OAuth**
5. Если система попросит настроить экран согласия:
   - Выберите **Внешний**
   - Укажите название приложения и свой email
   - Сохраните настройки
   - После этого вернитесь к предыдущему шагу
6. В качестве типа приложения выберите:
   - **Desktop App (Приложение для ПК)**
   - Нажмите **Создать**
7. Нажмите **Скачать JSON**
   - Переименуйте скачанный файл в `credentials.json`
   - Поместите его в папку проекта

---

## 3. Получение Telegram API

1. Перейдите на https://my.telegram.org и войдите в свой аккаунт.
2. Откройте раздел **API Development Tools**.
3. Создайте новое приложение.
4. Скопируйте значения `api_id` и `api_hash`.
5. Откройте файл `birthday.py` и укажите свои данные:

```python
api_id = 12345678
api_hash = "ваш_api_hash"
```

То же самое необходимо сделать в файле `get_id.py`.

---

## 4. Получение ID Telegram-канала

1. Создайте канал в Telegram.
2. Добавьте себя в качестве администратора.
3. Запустите скрипт:

```bash
python get_id.py
```

4. Найдите свой канал в списке и скопируйте его ID (например, `-1001234567890`).
5. Укажите этот ID в файле `birthday.py`:

```python
CHANNEL_ID = -1001234567890
```

---

## 5. Запуск

Запустите основной скрипт:

```bash
python birthday.py
```

### При первом запуске потребуется авторизация

#### Telegram (Pyrogram)

- Введите номер телефона.
- Введите код подтверждения из Telegram.
- Будет создан файл:

```text
my_account.session
```

#### Google Contacts

- В браузере откроется окно авторизации Google.
- Разрешите доступ к контактам.
- Будет создан файл:

```text
token.pickle
```

После этого дальнейшие запуски будут выполняться автоматически без повторной авторизации.

---

## 6. Обновление списка контактов

Если вы добавили новые контакты в Google Contacts, просто запустите скрипт повторно:

```bash
python birthday.py
```

Старые запланированные сообщения будут удалены, а новые созданы автоматически.

---

# Частые проблемы

### Ошибка: `Peer id invalid` или `ID not found`

Откройте канал в обычном клиенте Telegram под тем же аккаунтом, который используется для запуска скрипта, и повторите запуск.


### Скрипт не видит контакты

Проверьте, что у контактов заполнено поле дня рождения на сайте:

https://contacts.google.com

Если авторизация Google была сброшена:

1. Удалите файл:

```text
token.pickle
```

2. Запустите скрипт повторно.

### Браузер не открывается при авторизации Google

Скопируйте ссылку, отображаемую в терминале, и откройте её вручную в браузере.
